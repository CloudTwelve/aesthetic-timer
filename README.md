# Aesthetic Timer
A timer tab where you can choose the "aesthetic" of the page and the length of the timer.

## Features
- Custom duration of timer
- Custom timer names
- 6 different aesthetic themes
    - General Aesthetic
    - Dark Aesthetic
    - Cottagecore
    - y2k
    - Vintage
    - Soft Girl (coquette)
- To-do list/Goal setting
- Timer history
    - See/reuse past timers
